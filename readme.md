## convert markdown to bbcode

由于bbcode不支持行内代码，所以将其转换为粗体

## 在看番剧的iCalendar格式日历

`https://www.trim21.cn/api.v1/calendar/bgm.tv/{user_id}`

根据`https://api.bgm.tv/user/{user_id}/collection?cat=watching`生成，
所以不会排除已完结的番剧和未开播的番剧
